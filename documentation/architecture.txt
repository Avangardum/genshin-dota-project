Genshin Dota Project Consists of the following modules:
Genshin elements
Heroes (each hero is a separate module)

Genshin elements module
Contains all logic of elements and provides functions to apply elements and elemental
damage to units. 
Dependencies:
None

Heroes modules
Each hero is a separate module. Heroes shouldn't depend on each other
Dependencies:
Genshin elements module

Framework (everything can depend on it):
Dota lua API
Barebones libraries
Lualinq

Dependency graph:
Heroes -> GenshinElements
